量表说明

GFTI 四码

站点使用四码作为可传播标签，底层由 7 个连续维度映射而来。

  • 首位：C 主导 / O 外显 / R 保留 / F 冲击
  • 第二位：D 纪律 / A 适应 / R 保守 / V 变量
  • 第三位：A 亲和 / C 冷感 / R 理想 / F 爆发
  • 第四位：T 团队 / D 独行 / R 风险 / S 稳态

GFTI-Full 七维

  • 指挥性 Command
  • 保留度 Reserve
  • 纪律性 Discipline
  • 适应性 Adaptation
  • 温度感 Warmth
  • 队伍信任 Team Trust
  • 波动性 Volatility

每项 0–100，越高说明该特征越强。

使用边界

这不是官方人格设定，不代表角色“唯一正确解”；更适合作为二创、设定讨论、同人写作和角色分组的参考底盘。
